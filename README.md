# algorithm

競技プログラミング用ライブラリ

## cpp

以下に依存している部分がある

- https://github.com/google/googletest
- https://github.com/online-judge-tools/oj

基本コマンドは以下

```bash
# ~/cpp にいる想定

# main.cpp を in.txt 入力で実行
# main_resolved.cpp を提出
$ make

# 単体テストを実行
$ make test

# `Library Checker` のテストケースでテスト
# ntt は例
$ make oj@ntt
```
